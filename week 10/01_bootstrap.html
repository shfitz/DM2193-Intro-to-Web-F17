Hello, Bootstrap!

Today, we're going to talk about Bootstrap, arguably the most popular framework for building websites around. It contains HTM, CSS and javascript components to get you up and running with a website super quick.

While Bootstrap provides a lot of interactivity options, today we're going to focus on the layout aspects of it. We'll return to the other parts soon enough.

there are a couple default CSS styles that we'll look at. The interactiosn that come with javascript include

  • Alerts for dismissing
  • Buttons for toggling states and checkbox/radio functionality
  • Carousel for all slide behaviors, controls, and indicators
  • Collapse for toggling visibility of content
  • Dropdowns for displaying and positioning (requires Popper.js)
  • Modals for displaying, positioning, and scroll behavior
  • Navbar for extending our Collapse plugin to implement responsive behavior
  • Tooltips and popovers for displaying and positioning (requires Popper.js)
  • Scrollspy for scroll behavior and navigation updates
