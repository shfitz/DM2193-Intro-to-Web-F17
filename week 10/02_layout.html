Like our old friend 996-grid, Bootstrap is based on a responsive grid system. It's also (and this is pretty nice) a mobile first framework.

To use boostrap, you create a container div that wraps the content of your page. Inside, you'll place divs for rows, and then divs for columns. The number of columns in a row shoudl always equal 12.
