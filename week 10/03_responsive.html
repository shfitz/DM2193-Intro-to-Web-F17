Responsive layout!

A lot of websites have different looks between mobile, tablet, and desktop layouts.
On a big computer screen we may want a 12 or 6 column layout, then a 3 column for tablets and a 1 column for mobile phones.
Bootstrap takes this into account and has some built in CSS classes we can use.
.col-xs-# styles the layout for the smallest screens
.col-s-# styles the layout for small screens
.col-m-# styles the layout for the average sized screen
