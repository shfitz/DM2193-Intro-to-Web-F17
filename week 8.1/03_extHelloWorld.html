We can also link to an external file, just like an external stylesheet. This is useful when you want to use the same thing on multiple pages.

<!DOCTYPE html> 
    <html> 
        <head> 
        <title>Hello from out of this World</title>
        <script src="script.js"> </script>
    </head>
    <body>
    </body>
    </html>

When we do this, it's useful to recall that javaScript doens't have any native display functionality. things like the p5js canvas are built into libraries. we need to program our outputs.

we already say 

alert();
. Other things we'll use to display information include 
console.log();
which will log information to the browser's console window, and 
.innerHTML
which changes the text inside a HTML element. We'll get into mre specifics with that later.
