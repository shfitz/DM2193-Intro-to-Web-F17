Hello Javascript

Just like CSS, we can include javascript in the <head> and <body> of our pages.

<!DOCTYPE html> 
    <html> 
        <head> 
        <title>Hello World</title>
        <script type="text/javascript"> alert("Hello World!"); </script>
    </head>
    <body>
    </body>
    </html>

The 

alert();
function displays a pop-up with whatever data we put between the parentheses.
