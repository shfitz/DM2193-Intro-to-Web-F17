Accessing the DOM

There are two steps to access and update the DOM.

  1. Access the elements. With a DOM query, you can sleect an individual element node or multiple elements (aka a node list). You can also 'traverse the DOM' which involved moving a node from one element to a related one.
  2. Work with the elements. You can access/update text nodes, work wth HTML elements, and access/update attribute values.

DOM Queries

  • document.getElementById("IDname");
    will return a single element on the page with the ID you looked for (remember how we talked about only having one element with an id name?
  • document.querySelector(".container");
    will return the first element on the page that matches the CSS selector
  • document.getElementsByTagName("h1");
    will return an array of elements on the page that match the specified tag
  • document.getElementsByClassName(".active");
    will return an array of elements on the page that match the specified CSS class
  • document.querySelectorAll(".container");
    will return an array of elements on the page that matches the CSS selector
