Expressions

We all complain about math (except for mathmeticians and physicicsts, but i'm not sure they're realy human), but here's where we can put our educations to use.

javascript, like most programming langauges, has the ability to create mathematical expressions to evaluate numbers.

console.log(2+2); // writes 4 to the console

you can do a lot with math in javascript using algebra alone. setting varables, manipulating them, doing all sorts of funky shit (nb :funky shit will happen soon enough)

var a = 3;
    var b = 5;
    var c = a*b;
    console.log(c);

the above code snippet will write 15 to the console. You can it here.

the basic artithmetic operators are

  • + : addition
  • - : subtraction
  • * : multiplication
  • / : division
  • % : remainder (aka modulo, aka mod)
  • ** : exponetials
  • ++ : increment
  • -- : decrement
  • - : unary negation (turns somethig to its negative)
  • + : unary plus (changes string to integer)
For more information, check out this page chock full of examples.
