Overlapping elements

this is like, positioning text over an image

The z-index property specifies the order of an element (which element should be placed in front of, or behind, the others). Thsi value can be positive or negative. The higher an index value, the higher it appears on the stack. That is, soemthign with a z-index value of -1 will appear behind somehting with a value of 0, 1, 2, 3, etc.

If two positioned elements overlap without a z-index specified, the element positioned last in the HTML code will be shown on top.
