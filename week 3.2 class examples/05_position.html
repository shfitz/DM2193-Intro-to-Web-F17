The position property specifies the type of positioning method used for an element.

position values are

  1. static
  2. relative
  3. fixed
  4. absolute
  5. sticky (but not all browsers support it and it's sort of a bad user paradigm, so we'll skip it)
elements then have properties for left right top bottom. They work differently depending on what type of position value is being used.
