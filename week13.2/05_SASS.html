SASS

Like HAML, Sass (Syntactically Awesome StyleSheets) is a preprocessed language using Ruby, but it's for CSS. There's a related langage called SCSS (Sassy CSS) that has a more flexible syntax than Sass.

Like HAML, Sass and SCSS are compiled with Ruby and need to be installed via the command line with gem install sass. SCSS and Sass files need to have the .scss or .sass file extension.

To convert these files, you run sass style.sass style.css, similar to HAML. Unlike HAML, Sass allows you to watch for changes in source files, so it will compile automagically for you.sass --watch styles.sass:styles.css will make changes for you automatically based on changes to the source. You can watch a whole folder with sass --watch assets/sass:assets/css
