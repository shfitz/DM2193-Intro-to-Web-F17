variables

Holy smokes! Variables in CSS, at last! you can set a variable with $variable-name: variable. So you use a dollar sign, give the variable name, and finally the value to hold.


$border-style: dashed
$border-size: 1px
$sans-serif: "Verdana", sans-serif

p
    font: $sans-serif
    border:
        style:border-style
        size:border-size

Variables can be placed pretty much wherever in Sass, but if youre using it to name classes, ids, properties, or in a string, you'll need to use a slightly different syntax : 


$type: root
$offset: left

.#{$root}
  #{$offset}: 20px
