making HAML

HAML markdown is a bit different from HTML, but once you get the hang, it may be a bit easier. To declare the doctype as html, your HAML file should start with !!! 5.

Because the devs want tis to be beautiful, there's a hierarchy. the syntax is what really sets this apart. Unlike HTML, HAML only has an openig tag. They are initiated with a % abd indented to show off how they are nested. You can use tabs or spaces, but you must be consistent.

The "Hello World" of HAML looks liek this


!!! 5
%body
  %header
    %h1 Hello World
  %div
    %p Here's some text in a section

Text can be below the element if you like it that way too


!!! 5
%body
    %header
        %h1 
        Hello World
        %div
            %p 
            Here's some text in a section
