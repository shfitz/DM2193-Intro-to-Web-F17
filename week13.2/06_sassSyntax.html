Sass and SCSS Syntax

Each of these has a unique syntax. SCSS is pretty similar to regular CSS (you can write straight up CSS in there and it'll be ok). Sass, however, is pretty strict and poorly formatted code won't compile. Sass doesn't have any curly brackets or semicolons, just indentation.

SCSS example

.container {
    color: #ff7b29;
    font-weight: bold;
    p {
        text-transform: uppercase;
    }
}

CSS output

.container {
  color: #ff7b29;
  font-weight: bold;
}
.container p {
  text-transform: uppercase;
}

Sass Example

    color: #ff7b29
    font-weight: bold
p
