HAML

HAML, aka HTML abstraction markup language, is a markup language. The goal is to make "beautiful" markup. You write it one way, but it gets converted into HTML.

To use HAML, you need to have ruby installed on your machine. If you have OSX, it's already there. If you have Linux, it's probably there as well. If you have windows, you can install it here.

Once you're ready, in your terminal, type in gem install haml When you create HAML files, you need to use the .haml extension. To convert them to HTML, you use the command haml source.haml destination.html.

Admittely, this is best utilized in a shop that relies on Ruby, but it does allow you to see the semantic layout of your pges realy easily and quickly. You won;t forget a closing tag, and even if you don't actively use it, it's a god thing to be aware of. Let's write some HAML!
