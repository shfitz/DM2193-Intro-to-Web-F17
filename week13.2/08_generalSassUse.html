general use

nesting

You can nest selectors and properties inside of one another in Sass easily. With selectors, it would look like this : 


.container
  border: 1px solid #9799a7
  ul
    list-style: none
  li
    float: left

nesting properties is probably going to be more useful for you right now. you can do things like list out all the font, margin, padding, etc. properties like this : 


div
  font:
    family: sans-serif
    style: italic
    weight: normal
  border:
    style: dotted
    width: 1px
    radius: 20px

media queries

you can nest media queries as well.


.container
  width: 960px
  @media screen and (max-width: 960px)
    width: 100%

pseudo classes

you can add styles to a pseudo class with an &


a
  color: #0087cc
  text-decoration:none
  &:hover
    color: #ff7b29
    text-decoration:underline

comments

like HAML, there are general comments for CSS, and comments for Sass itself


/* CSS comment */
div
  background: #333
// Sass comment
h1
  font-family: san-serif
