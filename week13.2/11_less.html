Less

Less is a CSS preprocessor in javascript. you can run it from the command line through node.js, or you can run it in the browser by including the less.js scripts.

This page is using less. You can see the stylesheet in the head : <link rel="stylesheet/less" type="text/css" href="styles.less" /> . After you have imported the stylesheet, then you can import the less.js file from the CDN <script src="//cdnjs.cloudflare.com/ajax/libs/less.js/2.7.2/less.min.js"></script>
