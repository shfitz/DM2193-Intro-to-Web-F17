996 Grid

996 Grid is a CSS framework with a fixed-width made up of 12 even columns. This adapts content automatically to display nicely on different screen sizes from desktop to mobile. At the largest size, the content is allowed a 970px-wide area which can be divided into equal columns of two, three, four, or six.

CSS frameworks make it easy to get a site up and running with minimal hassle. They generally adhere to web standards and, generally, are in a grid of some sort. The CSS helps with layout, and oftenties there's a javascript component that can be leveraged as well, though the js bits are mostly concerned with layout and not interaction.

996 is robust and easy to use (hooray!) When you download it, you'll get some basic pages and folders.

  • A basic index.html file
  • an image folder
  • a css folder
  • a javascript folder named "js"

The CSS folder has a bunch of files in there already.

  • normalize.css : a file that 'normalizes' the css experience across browsers.
  • base.css : HTML5 Boilerplate for consistent design across browsers
  • grid.css : he heart of the grid system
  • style.css : Stylesheet for user styles. Includes a variety of useful media queries. This is what we'll be editing (though it can be useful to look through the others to know what's going on under the hood)

the js folder has a couple files in there. modernizr.js makes sure that browsers use the components that are relevant by detecting what features they support. the empty file is for us to dump stuff if we want to.
