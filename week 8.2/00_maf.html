built in Math functions

there' quite a bit we can do out of the box, but in order to do so, we need to use the Math object. It, like a lot of javascript, uses dot notation to access functions.

Math.floor

rounds down to the nearest integer

var myFloat = 3.33;
var myFloatToo = Math.floor(myFloat); // gives us 3

Math.ceil

rounds up to the nearest integer

var myFloat = 3.33;
var myFloatToo = Math.ceil(myFloat); // gives us 4

Math.round

rounds up or down to the nearest integer

var myFloat = 3.33;
var myFloatToo =3.66;
console.log(Math.round(myFloat)); // 3
console.log(Math.round(myFloatToo)); // 4

Math.random

gives a random number b/w 0.0 and 1.0

var myNum = Math.random();
console.log(myNum);

a fun trick to use Math.round with Math.random to give us a 1 or a 0

var yesNo = Math.round(Math.random());
console.log(yesNo);

let's flip a coin

What if we wanted a whole number randomizer? Easiest to make our own function that does such a thing. would look like this : 

function randomNumber(n) {
  var rn = Math.round(n * Math.random(n));
  return(rn);
 }

what's the crazy return doing in there? It sends the value back to the function that called it in the first place. it's how we get values from one place to another. In the above example, we'll get a number between 0 and n.

sometimes though, we want a random value between 2 arbitrary numbers, like soemthing in the range of 255 and 1024. This function will help with that : 

function randNumRange(min, max) {
 var rn = Math.round(Math.random() * (max - min) + min);
 return rn;
}
