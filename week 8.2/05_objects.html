Objects

Variables and arrays are neat! But sometimes we have different needs than just grouping things based on names or locations of images. Imagine three arrays like the below.

var catName = ["weirdo", "fatty", "chairman meow"];
var catColor = ["tortise", "black", "red"];
var catFood = ["tuna", "pepperoni", "hong shao rou"];

it would be a lot easier if we were able to refer to these elements as belonging to a certain cat instead of spreading them across three different arrays. That's where objects come in to play. You can combines different kinds of information inside of each object. With our cats, we would do it like this : 

var catOne = {name : "weirdo", color : "tortise", food : "tuna", age : 8};
var catTwo = {name : "fatty", color : "black", food : "pepperoni", age : 12};
var catThree = {name : "chairman meow", color : "red", food : "hong shao rou", age : 124};

this gives us variables that hold information in a more meaningful way (yay!). This also illustrates a super handy thing we'll be using on and off in the future and ever called key value pairs.

we access the inforamtion inside these through the dot notation we're already famiiar with. here's an example of gettig information out of an object
