Javascript Arrays

Arrays are great. Think of them like an egg carton in that they can hold many different things (in this case, the things are lists of values or objects). What's cool about this is that we can now have a variable that holds multiple items which we can access individually.

to declare and instantiate an array you would do so like this : 

var myNames = ["rev john", "chutney pants", "droolcup", "shfitz", "hey you", "bortrob"];
        var myNumbers = [10, -45, 3, 11, 465, -9642];

Objects and values are separated by commas, when we first create an array, we put the values inside square brackets [].
