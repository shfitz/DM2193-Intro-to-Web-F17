Geolocation

In order to do geolocation, you need to be running a server. There are a couple different ways to do that. If you're running Brackets, it's as easy as clicking the lightning bolt on the side of your editor. Sublime text has a package called "Sublime Server". If you're using something else, or want a challenge, you can use node.js. mas ship with python. you could start a server from the terminal in the directory of your choice with python -m SimpleHTTPServer 8000
