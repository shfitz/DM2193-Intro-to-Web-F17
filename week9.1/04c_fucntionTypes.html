Many kinds of functions in javascript

We've seen traditional function declaration like this : 

function roomSize(width, height){
    return area = width * height;
}

var space = roomSize(10, 10);

but there are other ways of declaring and using functions. one is to use it as an expression. The name is typically omitted. This is also called an anonymous function. This is used when code needs to run once inside a task. 

// function as an expression
var area = function(width, height){
    return area = width * height;
}

var space = area(10, 10);

You'll run into this alot in jQuery (coming up next!) so it's a good FYI. It's often used as an argument inside another function.
