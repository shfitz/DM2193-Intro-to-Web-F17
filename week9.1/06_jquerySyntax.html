Jquery Syntax

Everything in Jquery is done through the Jquery object. We looked at objects last week as a data storage tool. Jquery is a little different in that it stores data and has functionality. it's like getting a free gift froma mail-order catalog.

firing up jquery

there's a base function called jQuery() that lets you find elemensts in a page. As lazy programmers, we hate writing out jQuery() all the time. fortunately, there's the option of of using $() instead.

in javascript, we would type <body onload="myFunction();"> to get something to happen after the page loaded. Jquery has a built in function for this, which helps make sure there's no issues with trying to access DOM elements before the page finishes loading. You can wrap all your code inside this, which makes it convenient for sticking in the head of your pages.

$(document).ready(function() {
  // your code goes here
});

Honest to goodness use

in Jquery, there are some similarities to the DOM queries we've been using, but it's a bit simpler. The syntax for calling jquery is 

$(selector).action();

the selector above is the query for HTML elements. the action() is the thing to do to the returned element/s.

There ae some selectors that will get more play than others. However, they are all similar to the ones you're accustomed to from CSS

// javascript
document.getElementById("myID");  
// jquery 
$("myID") 
// returns a single element with the searched id
        
// javascript
document.getElementsByTagName("p");  
// jquery 
$("p")
// returns an array of elements from the page with the specified tag
        
// javascript
document.getElementsByClassName(“.coolness”); 
// jquery 
$(".coolness")
// returns an array of elements from the page with the specified css class

// javascript
document.querySelector(".coolness");  
// jquery 
$("li.coolness")
$(".coolness:first")
// returns the first element on the page that matches the CSS selector searched for. 
// use :first to find the first element that matched either the tag or class.
