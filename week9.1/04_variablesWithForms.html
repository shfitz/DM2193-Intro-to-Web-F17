Other kinds of inputs

text areas

You can define a multi-line input field for text input with textarea

<textarea name="response" rows="10" cols="30"> 
        Please add your thoughts here 
        </textarea>

checkboxes

Allow a user to toggle things on and off

<form>
 <input type="checkbox" id="turnBrightOn" name="brightness" value="lightOnOff">
 <label for="brightness">Check the box to turn the lights on</label>
 </form>

dropdowns

the select element allows you to choose from a dropdown. the option element identifies the varuous choices

<form> 
        <select name="catNames"> 
          <option value="weirdo">Weirdo</option> 
          <option value="fatty">Fatty</option> 
          <option value="chairmanMeow" selected>Chairman Meow</option> 
          <option value="pepperoni">Pepperoni</option> 
        </select>
</form>
