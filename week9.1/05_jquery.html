Jquery

What the heck is jquery?

Jquery is a javascript library that makes it easy to do stuff with javascript in a website. Last week, you were using regular js for basic functionality, but it can be a little daunting to deal with all the funny syntax for accessing elements in the DOM. Jquery fxes that by introducing a new syntax (yay?) that stremlones the process of accessing and changing elements in a site.

Eseentially, Jquery makes it so you only have to write code once that will work in all browsers. It allows you to select elements with CSS style selectors, perform tasks like update the DOM tree, animate elements, and loop through arrays of elements in less code than regiular js, and handle events enabling you to attach event listeners to elements withouth haveing to muck around with code for older browsers.

Like any other external js file, we want to link to jquery's library in the head of the page. You can either host a version of the library locally on your server, or you can link to a CDN like this : 

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>

An upside to using a cdn for your sites is that the content will cache in your browser, and if it's hosted by someone like google or microsoft, they have fat pipes that will help get the file to a client's browser more quickly.
