local servers

These are largely based on the p5js local server notes.

It can be really useful to run a local server soemtimes for checking things. There are a number of ways to do this depending on the platform you're using. In class, I've been using the live preview capabilities of Brackets. It starts a server on my computer that coughs up webpages when I make requests. Sublime Text has a plugin for this as well.

Python

On OSX you already have python installed. Linux and windows folks can install from Python.org. You can start a server with the linepython -m SimpleHTTPServer in the terminal. This will start a server with the root being whatever directory you started the server from. It's a little pokey, but shoudl suffice for most uses.

node.js

node is a javascript library that is server-side, not the client side stuff we've been using. That is, it's an independent thing on a computer, it doesn;t run in the browser. It's faster than the python serer, becoming more widely used every day, and is the hip thing to do. Plus, you already know javascript, and darn it, that's worth something.

  1. Download and install node.js for the platform of your choosing.
  2. in the terminal, type (Linusx/macOS) sudo npm install -g http-server and enter your password. In windows, open the command prompt and type npm install -g http-server (you will probably need to be an admin for this.)
  3. type http-server
npm is shorthand for "node package manager". It's a tool that allows you to quickly download and use packages that extend the code node library.

Apache

Apache is one of the oldest an most widely used servers on the web. on the mac, it's already installed. you can start it by typing sudo apachectl start in your terminal. Visit localhost in your browser to verify it's working. The server root is at /Library/WebServer/Documents. You'll need to change permissions on the directory to addd items to it. For windows, download WAMP, install it and follow the instructiosn to set up the server. On Linux (Ubuntu), install with sudo apt-get install apache2. The server root is at /var/www/. This Linode documentation is pretty good for setting up a server on Linux.
