APIs

API is short for Application Program Interface. For us, it's an esy way we can access features of websites and browsers. More generally though, its how we use different programming environments and langauages. APIs can be public or private.

Web APIs allow you to use javascript to interface with a page or the browser. Here's a complete list of Web APIs. These allow you to use stuff the browser offers you. We're going to start with using browser APIs for geolocation, though we've already seen them in use last week when dealing with local storage.

To make the geolocation stuff happen though, we need to run a local server. For security purposes, browsers restrict access to some types of information unless it comes from a server.
