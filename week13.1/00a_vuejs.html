Vue is a frontend javascript framework for building user interfaces. You can use it in conjunction with backend systems, but it doesn't require them. it's similar to React (which we'll get to later). When you first start with Vue, you have the choice of using a command line version of it, which compiles on the server, or the browser based javascript way you're accustomed to. For the time being, stick with what you know.

Vue (and react) uses a "virtual DOM", which means that you can quickly update elements in your page without having to refresh to reload. It's set up so data can be rendered a template, which gives you a lot of flexibility to make things happen in the browser.
