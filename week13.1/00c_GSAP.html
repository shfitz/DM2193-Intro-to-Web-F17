GSAP

Greensock Animation Platform is one of the more popular libraries for animation with javascript. There are a couple different libraries in there : "TweenLite","TweenMax", "TimelineLite" and "TimelineMax".

TweenLite

include the following in your page :

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/1.20.2/TweenMax.min.js"></script> and 

<script>
window.onload = function(){
    var logo = document.getElementById("cat");
    TweenLite.to(cat, 10, {left:"632px"});
}
</script>

The first blub of code will load the library in the page. The second one will tween the image from the left side of the window to the right over one second. The syntax is TweenLite.to( [target object], [duration in seconds], [destination values] )

You can customize the kind of easing in the tween as well. There are many, many types. Here's an interactive widget for seeing tweens.
