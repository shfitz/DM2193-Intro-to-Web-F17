React.js is a javascript library for building user interfaces. It's made by facebook. Like vue.js, it uses a virtual DOM to render things to screen. It is built with javascript, but has this funny thing named JSX that's not necessary to use, but it generally helpful.

The basic building blocks are called components, and these render to an instance of react that is bound (what they call mounted) to an element, typically a div that holds the entire page.
