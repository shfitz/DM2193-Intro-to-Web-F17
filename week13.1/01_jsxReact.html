JSX is a funny thing. It looks like this : 

    
<script type="text/jsx"> 
    /** @jsx React.DOM **/ 
    React.render( 
        <h1>Hello, world!</h1>, 
        document.getElementById('myDiv') 
    ); 
</script>

JSX is short for javascript XML, which lets you write HTML inside your javascript. It's not really HTML though, it's actually XML. The one notable difference is that the class attribute needs to be written as className. Here's a complete list of "gotchas". You don't have to use JSX though, you can also write in plain old javascript, but it looks funny when you do. 


/** @jsx React.DOM **/
React.render(
  React.DOM.h1(null, 'Hello, world!'),
  document.getElementById('myDiv')
);

When using the render method, the first argument is the component we want to render, the second is the element it should mount to.
