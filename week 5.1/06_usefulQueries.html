media query types

  • all Used for all media type devices
  • print Used for printers
  • screen Used for computer screens, tablets, smart-phones etc.
  • speech Used for screenreaders that "reads" the page out loud

media query features

  • There are many many features, a stupid amount to completely list here. What follows are the ones that will probably be most useful for now.
  • width describes the width of the targeted display area of the output device. min-width and max-width are part of this.
  • height describes the height of the targeted display area of the output device.
  • device-width describes the width of the rendering surface of the output device
  • device-height describes the height of the rendering surface of the output device
  • orientation returns ‘portrait’ when the value of the ‘height’ media feature is greater than or equal to the value of the ‘width’ media feature; otherwise, it's ‘landscape’.
